## 🧱 About PARS Network

**PARS Network** is a development team focused on creating powerful, optimized, and reliable software for **Minecraft: Bedrock Edition**.

Our main focus is **PocketMine-MP**, **PHP development**, server systems, APIs, libraries, and tools for the Minecraft Bedrock ecosystem.

> ⚡ **Build. Optimize. Deploy.**
